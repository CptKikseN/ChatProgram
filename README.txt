Project name:
ChatProgram2.0

Description: 
The program “ChatProgram2.0” is chat with a single server that can manage multiple clients. ChatProgram2.0 uses the internet protocol TCP (Transmission Control Protocol). 

Installation:
The program do not require any installation, but the user needs to ensure that the ip address of the clients are the same and that the port number in both the client and server is the same.

Usage:
Type any desired text and communicated with other clients.
